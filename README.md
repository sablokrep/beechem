# beechem

- a single crate for the classification of the bees chemical expression to sequence using all the classifier, regressor, and the neural networks.


- chemical modelling of bee glues
- chemical modelling and Quantitative Structure Activity Relationship (QSAR) of bee glues. 
- chemical composition associated with gene expression using a elasticnet approach. 

- Chemical classifier based on 200 chemical smiles properties
- added the smiles to sequence and gene expression and a complete logistic classifier as chemicals are treated as a single experiment.
